Multi-Sensor Perception Pipeline

Parsa Ghasemi1, Behnam Bahr1
1Cal Poly Pomona
Part of DRIVE Platform
AVROS

Multi-threaded sensor fusion, Bayesian occupancy grid mapping, and bird's-eye view perception for real-time autonomous navigation.

Abstract

We present a multi-sensor perception pipeline for a full-scale autonomous vehicle. The system fuses data from a Velodyne VLP-16 LiDAR, Intel RealSense RGB-D camera, and Xsens MTi-680G IMU at 100 Hz with thread-safe ring buffers and quaternion-to-Euler transformations. A probabilistic Bayesian occupancy grid with log-odds representation and Bresenham raycasting provides 0.2m resolution mapping over 400m² coverage. Bird's-eye view perception using inverse perspective mapping enables real-time camera-to-ground-plane transformations at 15–30 FPS. An Extended Kalman Filter provides GPS/IMU fusion with graceful degradation to IMU dead reckoning during GPS dropout.

Sensor Fusion Pipeline

The sensor fusion pipeline processes Xsens MTi-680G data at 100 Hz using multi-threaded architecture with thread-safe ring buffers. Quaternion-to-Euler transformations convert IMU orientation data for integration with GPS positioning. An Extended Kalman Filter fuses GPS and IMU measurements, with graceful degradation to IMU dead reckoning when GPS signal drops out—maintaining continuous pose estimation even in GPS-denied environments.

Occupancy Grid Mapping

A probabilistic Bayesian occupancy grid uses log-odds representation for numerically stable belief updates. Bresenham raycasting efficiently traces sensor beams through the grid, clearing free space and marking obstacles. Temporal decay enables handling of dynamic obstacles—cells gradually return to unknown state when not actively observed. The grid achieves 0.2m resolution over 400m² coverage area, providing the planning layer with a reliable local map for obstacle avoidance.

Bird's-Eye View Perception

The bird's-eye view (BEV) perception system uses inverse perspective mapping (IPM) to transform camera images from the perspective view to a ground-plane view at 15–30 FPS. This transformation provides a top-down representation that simplifies lane detection and drivable area estimation, complementing the LiDAR-based occupancy grid with visual road structure information.

Sensor Integration

The perception system integrates a Velodyne VLP-16 LiDAR for 3D point clouds, Intel RealSense RGB-D cameras for depth and color imagery, USB cameras for additional viewpoints, and an Xsens MTi-680G IMU/GPS for positioning. A unified sensor abstraction layer provides consistent interfaces across all sensors, enabling modular development and easy integration of new sensor types.
