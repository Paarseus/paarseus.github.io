Gymnasium-Compatible RL Training Environment for CARLA

Parsa Ghasemi1, Alexander Assal1
1Cal Poly Pomona
carla-rl-suite

A modular Gymnasium wrapper around CARLA that hits 100 steps/sec with sub-10 ms latency and zero memory leaks.

Abstract

We present a modular Gymnasium-compatible environment for the CARLA autonomous driving simulator that achieves 100 steps/sec training throughput with sub-10ms multi-sensor latency. The system uses a manager-based architecture with 7 modules and factory pattern sensors, enabling JSON-configurable training scenarios with dual-queue event handling. The environment supports continuous control action spaces (2D: steering + acceleration/braking) compatible with PPO, SAC, and TD3 algorithms. A PPO agent trained in this environment achieves stable lane-following behavior within 50,000 steps, validated over 10,000 episodes with no memory leaks.

Manager-Based Architecture

The environment is organized into 7 specialized managers: World Manager, Vehicle Manager, Sensor Manager, Observation Manager, Reward Manager, Scenario Manager, and Event Manager. Each manager encapsulates a specific concern, and a factory pattern enables dynamic sensor creation based on JSON configuration files. Dual-queue event handling separates synchronous simulation events from asynchronous sensor callbacks, ensuring deterministic training behavior.

Training Pipeline

The training pipeline achieves 100 steps/sec throughput by minimizing Python-CARLA communication overhead and batching sensor queries. Multi-sensor latency stays below 10ms through careful synchronization of camera, LiDAR, and state vector observations. Memory usage remains flat across 10,000+ episodes, verified through systematic leak testing. The environment implements standard Gymnasium reset/step interfaces for seamless integration with Stable-Baselines3 and other RL frameworks.

Observation & Action Spaces

The observation space combines RGB camera images, vehicle state vectors (speed, acceleration, steering angle), and navigation data (waypoints, distance to goal) into a multi-modal input suitable for end-to-end learning. The continuous action space maps a 2D vector to steering angle and combined acceleration/braking, compatible with on-policy (PPO) and off-policy (SAC, TD3) algorithms. Reward shaping balances lane-keeping, progress toward waypoints, and collision avoidance.

Results

A PPO agent trained for 50,000 steps achieves stable lane-following behavior on the CARLA Town01 map. The agent maintains center-lane driving through curves and intersections with minimal lateral deviation. The configurable scenario system enables evaluation across different towns, weather conditions, and traffic densities through JSON configuration files without code changes.
